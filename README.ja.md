# オープンイベントさいたま

埼玉県の広報情報オープンデータを利用したイベント情報アプリです。

## 機能
- 埼玉県のイベント情報の表示
- カテゴリ別のフィルター
- 検索機能

## 必要環境
特に必要ありません。

## 使い方
- 表示されているイベント情報を閲覧できます。
- カテゴリをクリックすることで、カテゴリ別にイベントをフィルタリングできます。
- 検索窓に文字を入力すると、イベント名や説明文で絞り込めます。
- イベントカードをクリックすると、詳細情報を確認できます。

## データ・API
- 埼玉県オープンデータポータル「【埼玉県】広報情報」のオープンデータを使用しています。
  - https://opendata.pref.saitama.lg.jp/data/dataset/9b283d97-f53e-4346-b734-b43695eac1c9
